 # Moonlight Mayhem - Issues & Feature Requests

  Moonlight Mayhem is a free online social deduction party game inspired by classics like One Night Ultimate Werewolf. One player hosts the game on a TV or shared screen, while everyone else joins from their phones.

  **Play now at [moonlightmayhem.com](https://moon.gavindraper.com)**

  ## Reporting Issues

  Found a bug? Something not working as expected? [Open an issue](https://github.com/G-DigitalGaming/MoonlightMayhem/issues/new) and include:

  - **What happened** - Describe the problem
  - **What you expected** - What should have happened instead
  - **Steps to reproduce** - How can we recreate the issue
  - **Device/Browser** - What you were playing on (e.g. iPhone Safari, Chrome on Android)
  - **Screenshots** - If applicable, add screenshots to help explain the problem

  ## Suggesting Features

  Have an idea for a new role, game mode, or improvement? We'd love to hear it! [Open a feature request](https://github.com/G-DigitalGaming/MoonlightMayhem/issues/new) with:

  - **Description** - What's your idea?
  - **Why** - How would this improve the game?
  - **Details** - Any specifics on how it could work

  ## Note

  The game's source code lives in a private repository. This repo exists solely for public issue tracking and feature requests.
